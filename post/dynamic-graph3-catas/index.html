Dynamic structure network 3 -- Catastrophic forget

2020-08-08

The first time I start pay attention to Dynamic Graph network was I try to find a way to solve the catastrophic forget problem in continu learning(or class-increment learning).
The catastrophic forget is when continu learning begin and the data of new class input, the network will forget the old class. It's becouse the softmax function is more bias to the new class. That's why when we train a deep learning model we should mix all class together but can't just put in as queue.So many paper try to find a good way to mix the data of old class and new class. Most of them is try find the feature of old class than mix with new class. I have to say this way is useable but it just treat the symptoms but not the root cause.
I think the root of this problem is the neural. When the new class data input, the neural will start updata the parameter and after that, this neural's parameter will lost the relationship with the old class. So we have to find a methed to keep the information of old class. In 2020 a paper have solve this problem with a study of animal's nervous system.
In this paper, following the logic of the elastic weight consolidation method (EWC), each connection weight wiw_iwi​ in the NN is matched by its “significance” for the skill acquired during training on the first training set A. The “significance” of the weight w i shows how much the change of the weight wiw_iwi​ will be penalized when training on the next training set B. That is, a change in the weight of w i is penalized the more, the more its “significance” is.
So, this paper give the loss function as

L=LB+λ2∑Fi(wi−w∗A,i)2L = L_B + \frac{\lambda}{2}\sum{F_i(w_i - {w^*}_{A, i})^2} L=LB​+2λ​∑Fi​(wi​−w∗A,i​)2

Then it replace the FiF_iFi​ (fisher information matrix) with SwA,i{S^w}_{A, i}SwA,i​ and SbA,i{S^b}_{A, i}SbA,i​ this two thing is:

SwA,i=1n∑∣xk,iwA,i∣{S^w}_{A, i} = \frac{1}{n}\sum{|x_{k, i} w_{A, i}|} SwA,i​=n1​∑∣xk,i​wA,i​∣

SbA,i=1n∑∣yk,i∣{S^b}_{A, i} = \frac{1}{n}\sum{|y_{k, i}|} SbA,i​=n1​∑∣yk,i​∣

So, the loss function become:

L=LB+λ2∑SwA,i(wi−w∗A,i)2+λ2∑SbA,j(bj−b∗A,j)2L = L_B + \frac{\lambda}{2}\sum{{S^w}_{A, i} (w_i - {w^*}_{A, i})^2} + \frac{\lambda}{2}\sum{{S^b}_{A, j} (b_j - {b^*}_{A, j})^2} L=LB​+2λ​∑SwA,i​(wi​−w∗A,i​)2+2λ​∑SbA,j​(bj​−b∗A,j​)2

After that, they update the w′i{w'}_iw′i​ and b′j{b'}_jb′j​ become:

w′i=wi−α1+λSwiL˙wi˙{w'}_i = w_i - \frac{\alpha}{1 + \lambda {S^w}_i}\frac{\dot{L}}{\dot{w_i}} w′i​=wi​−1+λSwi​α​wi​˙​L˙​

b′i=bi−α1+λSbiL˙bi˙{b'}_i = b_i - \frac{\alpha}{1 + \lambda {S^b}_i}\frac{\dot{L}}{\dot{b_i}} b′i​=bi​−1+λSbi​α​bi​˙​L˙​

This paper is work on static network and our work is on dynamic network but we can still base on this.
We can see in this way we can keep the plasticity of our network and no need for initiative forget. So we can say this function is very neture.

Dynamic graph network 4 -- What kind of struct should we use ?

2020-08-10

This article will discussion about the struct of our dynamic network.
I think the network can simple divid into three part:

  1. The input block
  2. The memory block
  3. The output block
    As for the type of neuron I said in the previous article, we temporary confirm there will only one variety. Becouse make the neuron different is bad of the expandability.
    About the three block of our network:
  4. -- The input block
    This block will used to deal with the diffierent kinds of data, extract there feature, than put them(the feature) into memory block to learn. And this part will have a dynamic connect with the memory block, when a node of memory block have a high "vitality", it will completely not connect with input block.
  5. -- The memory block
    This block is the main part of our network, the "dynamic" mostly means this block have a regular change. Becouse this part is maintain the memory function of network, just like the cerebral cortex(Just maybe, I have to say I'm not a good biology student).
  6. -- The output block
    This block is just like the traditional neuron network, but it can add new node of new class.

This article will just a simple intorduction of the dynamic network sturcture, we will make a detailed discussion in another article.
And so far, the problem in the second article all have a general outline.

Dynamic structure network 6 -- How should we initialize our network?

2020-09-18

As last artical said, the simple initialize structure will made network overfitting when structure adaptive training. But it obvious, if the first time training make our network overfitting, the increase training will run into a great obstruction.
So now we have to find a way to overcome the overfitting during training or give our network a complex initialize structure.
For my opinion, we can have them both.

(I want them all !!)
Now we can split the training process into two part. The first called "Baby Training" and the second part called "Growing Training".
Baby Training's main job is training with the Dataset1. This part will just like Cascade-Correlation network. Before training start, we only get input layer and output layer. After training, we random remove some of the connect. Then we will get the initialize structure of our network.
Until now, our training process is clear. We will discuss some detail on next artical.
